# 数据收集与变量对照表（查数据专用）

> **💡 数据收集建议**：
> 1. **优先收集“需要收集的基础原始数据”列的绝对值/绝对量**。
> 2. 所有带 `_ln`（对数）、`_share`（占比）、强度的衍生工作，将由代码全自动计算完成，**不要在Excel中手动计算**，防止出错。
> 3. 如果某项指标缺失个别年份（如2011、2023缺失），代码会自动进行插值；但如果某指标缺失超过3-5年，建议直接放弃该指标。

---

## 1. 基础主键与计算底座数据
*(必须最先收集齐，用于计算各类“人均”、“强度”和“对数”)*

| 需要收集的基础原始数据 | 中文含义 | 最终代码内入模变量名称（衍生后） |
| :--- | :--- | :--- |
| `province` | 省级行政区名称 | `province` (例如：北京、江苏，需统一名称) |
| `year` | 年份（2011-2023） | `year` |
| `gdp` | 地区生产总值（实际值/名义值，请保持统一） | `gdp` |
| `population` | 常住人口数 / 年末人口数 | `population` |
| `co2_total` | 地区二氧化碳排放总量 | `co2_total_ln` / `carbon_intensity_ln` |

## 3. 控制变量原始数据
*(用于剥离其他宏观因素对碳排放的影响。只需收集**绝对量**，除法由代码做)*

| 最终控制变量意义 | 需要收集的原始数据 | 中文含义 |
| :--- | :--- | :--- |
| 人均 GDP | `gdp` & `population` | 见表1底座数据 |
| 第二产业占比 | `ind2_value_added` | 第二产业增加值 |
| 城镇化率 | `urban_population` | 城镇人口数（与总人口相除获得） |
| 对外开放度 | `total_import_export` | 进出口总额（注意汇率或直接用人民币计价） |
| 环境治理强度 | `env_gov_investment` | 工业污染治理完成投资额 / 环保支出 |
| 人口密度 | `land_area` | 地区行政辖区面积 |
| 煤炭消费占比 | `coal_consumption` & `total_energy_consumption` | 地区煤炭消费量与能源消费总量 |

## 5. 补充层 / 案例层（江苏/南京/苏州专属）

| :--- | :--- | :--- |
| `city` | 城市名 (江苏省、南京市、苏州市) | 区分观测值 |

| `pm25` | PM2.5 年均浓度 | 环保治理直观结果 |
| `good_air_days_rate` | 空气质量优良天数比率 | 环保治理直观结果 |
| `express_volume` | 快递业务量 | 反映流通与服务业的数字化用能特征 |
