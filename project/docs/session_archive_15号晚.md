## 待办项与全局质检优化方向 (下个对话目标: Stage 5 优化重构)
在进入正式论文撰写前，我们决定进行一轮**全局复盘与数据提纯**，以解决前期妥协带来的统计瑕疵并向顶刊标准冲击。

### 1. 数据面板与指标扩容（用户洞察）
- **充实数字基建底层指标（解决指数量化单薄问题）**：目前组成“能源数字化指数”的硬指标不够丰富。计划引入其他维度的替代数据（如：通过 Python 爬取各省政府工作报告中“智慧电网/能源数字化”的文本词频、能源上市企业数字化投入的省份汇总等）来让综合指数更加丰满和有说服力。
- **拓展时间维度面板T（解决年份截断问题）**：前期因为5G、绿证等“新概念”在历史年份（如2011-2015）缺失严重，被迫缩小了时间窗口。计划运用更高级的面板数据插补技术（如多重插补法 MICE、或者基于 ARIMA 的逆向期前推延），将时间跨度强行拉长，极大增加全样本量（扩大自由度，使得传统计量结果更易显著）。

### 2. 因果推断与模型升维（AI 专家补充方案）
- **内生性检验：引入工具变量 (Instrumental Variable, IV)**：目前的双向固定效应无法彻底根除“反向因果”——即“越是富裕且力求减排的省份，越有钱去铺设数字基建”。下一步必须加入 IV 检验（例如：常用的由“各省1984年历史邮电局数量”乘以“全国互联网接入端口数趋势”构造的经典工具变量，或基于地形坡度的光缆铺设地理阻力），采用 2SLS 两阶段最小二乘法进行更铁血的因果论证。
- **空间计量模型 (Spatial Econometrics)**：前期模型跑出 Driscoll-Kraay 空间标准误后，已经实锤了“碳排放和数字化都具有跨省溢出效应”。既然如此，不如直接将其升级为**空间德宾模型 (SDM)** 或添加空间权重矩阵（如地理距离矩阵/经济矩阵）。这既是一个巨大的学术发力点，还能衍生出“本省能源数字化对邻省碳排放的影响”的新故事。
- **机制深挖：调节中介效应**：不要局限于直接的乘积和基础分布，可尝试多重或链式中介响应。
