# 能源数字化项目 - 第四阶段总结与变量体系全景字典

本文档为项目的第四阶段（机器学习预测与可解释性分析）总结，同时作为全篇论文写作的核心参考材料，囊括了底层的变量体系、SHAP图表分析结论以及专业的学术术语。

---

## 第一部分：核心变量体系与模型构成字典

在复杂的数据分析和建模项目中，将所有的变量名进行结构化归类，明确它们在统计学上的角色（Y、X、控制变量、中介变量等）。

### 1. 核心回归变量体系 (模型入参)
*   **`carbon_intensity_ln`**（主因变量 Y）：碳排放强度对数（$ln(总碳排/GDP)$），反映经济增长的“绿色含金量”。
*   **`carbon_per_capita_ln`**（替代因变量 / 用于稳健性）：人均碳排放对数。
*   **`co2_total_ln`**（替代因变量 / 用于稳健性）：绝对碳排放总量对数。
*   **`energy_digital_index`**（核心解释变量 X）：能源数字化指数（根据三大维度降维合成的出厂原值）。
*   **`energy_digital_index_100`**：放缩到 0-100 区间的基准百分制指数，使回归系数更直观易读。
*   **`energy_digital_index_pca`**：应用主成分分析(PCA)合成的验证指数（用于稳健性检验）。

### 2. 传导机制与中介变量 (Mechanisms, M)
用于回答“能源数字化是**如何**促成减排的”：
*   **渠道1：能源结构优化** -> `coal_share` (挤出煤炭占比), `clean_power_share` (提升清洁电力消纳)。
*   **渠道2：绿色技术创新** -> `green_patent_ln` (绿色专利授权量)。
*   **渠道3：产业升级与能效** -> `tertiary_secondary_ratio` (三二产业产值比), `energy_intensity_ln` (单位能耗)。

### 3. 控制变量 (Controls / Confounders)
*   **`pgdp_ln`**：人均 GDP 的自然对数（控制经济发展水平的规模效应）。
*   **`ind2_share`**：第二产业增加值占比（控制重工业和加工业的产业结构）。
*   **`urban_rate`**：城市化率（控制人口集聚带来的耗能规律变化）。
*   **`coal_share`**：煤炭消费占比（控制各省不可改变的原始化石能源禀赋）。

### 4. 综合指数的底层数据画像 (三大维度)
能源数字化指数由以下统计抓取指标构筑：
*   **维度A：数字基础设施**：`broadband_users` (宽带用户), `5g_base_stations` (5G基站数), `optical_cable_length` (光缆线路长度)等。
*   **维度B：能源数字应用**：`charging_infrastructure` (充电桩基建), `electricity_market_volume` (电力市场化交易量), `green_power_trade_volume` (绿电交易), `renewable_installation_share` (可再生装机比例)。
*   **维度C：数字产业支撑**：`software_revenue` (软件业务收入), `digital_core_industry_share` (数字核心产业GDP占比), `rd_intensity` (研发强度)等。

## 第二部分：学术写作核心术语库 (Terminology)

建议在撰写论文与答辩时，将以下高频专业术语与结果对应：

*   **双向固定效应 (TWFE)**：吸收“省份特质”（地理气候）和“年份冲击”（宏观经济周期）的面板因果推断底座。
*   **非线性映射 (Non-linear Mapping)**：变量对结果的拉动具有拐点的曲线响应关系，由 XGBoost/RandomForest 捕捉。
*   **边际贡献 (Marginal Contribution)**：具体特指某个特征在控制其他特征后，使得碳排放偏离基线均值的纯粹预测拉动力（即数值意义上的 SHAP 值 $\phi$ ）。
*   **门槛效应 / 拐点效应 (Threshold Effect)**：当某指标（如数字化指数）只有越过特定数值区间，才会引发“量变到质变”的新功效。
*   **网络外部性 (Network Externalities)**：解释门槛效应背后的经济学逻辑。即数字网络必须铺设形成一定规模，才能产生实质的赋能收益。
*   **碳反弹期 (Carbon Rebound)**：数字化建设初期，由铺设 5G 或高耗能数据中心导致的碳排放短暂攀升现象。
*   **局部可加性 (Local Additivity)**：微观瀑布图的基础，解释省份自身优劣势如何通过代数相加以达到最终排放。

## 第三部分：机器学习与SHAP核心图表深度解析

结合 `project/outputs/figures/` 下输出的五张图表，核心学术结论如下：

### 1. XGBoost/RF 实际与预测对比图 (`*_actual_vs_predicted.png`)
绝大多数散点紧密地贴合在 $y=x$ 对角虚线上。此图证明了集成树模型捕捉到了面板数据中复杂的非线性映射关系，展现出了极高的预测优度（$R^2 > 0.8$），为后续提取 SHAP 解释提供了一个极具公信力的高维基座。

### 2. SHAP 全局特征重要性摘要图 (`shap_summary_plot.png`)
*   **影响力排序**：`coal_share`（煤炭依赖）和 `pgdp_ln`（经济发展水平）是第一梯队核心影响因素，而我们的核心变量 **`energy_digital_index_100` 稳居第三**。
*   **正负向影响**：红点（数字化指数高的样本）绝大部分聚集在纵轴左侧（SHAP < 0），蓝点（低指数）偏右，直观定性地证明了：**能源数字化水平越高，显著拉低碳排放强度**。

### 3. 能源数字化边际效应依赖图 (`shap_dependence_energy_digital.png`)
**本研究最具爆发力的发现图。**
*   **碳反弹阵痛期**：数字化指数在 `0.00` 到 `0.40` 区间时，SHAP 值为正。对应了“数字化基建阵痛期”——早期建设数据中心本身会拉升碳排放。
*   **跨越拐点**：当跨过 `0.40~0.50` 的门槛后，SHAP 值直接跌破 0 且深度下探。代表智能电网调度效率提升等网络外部收益彻底抹平了基建耗能，迸发出急剧的**碳减排赋能效应**。

### 4. 煤炭依赖度边际效应依赖图 (`shap_dependence_coal_share.png`)
呈现经典的**“曲棍球棍形状 (Hockey Stick)”**。
*   `coal_share` 在 `0.0` 到 `0.6` 爬升时，碳排增量走势平缓。
*   一旦越过关键的 **70% 煤炭占比红线**，碳强度的 SHAP 会向空中爆炸式飙升，证明了过度依赖煤炭带来的碳排代价是指数级而非线性的。

### 5. 局域归因微观瀑布图 (`shap_waterfall_single_sample.png`)
从微观视角解剖特定省市的碳排归因。利用 SHAP 的“局部可加性”，清晰展现出：极高的人均 GDP 和极低的二产占比会将碳排预期（蓝色箭头）拉至极低；但在数字化尚未越过门槛的个别年份，其微弱的数字基建水平反而成为唯一拖后腿的排碳因素（红色箭头），生动还原了现实中的减排约束博弈。
